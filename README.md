# Тестовое задание

В репозитории находятся решения пяти блоков тестового задания по анализу данных:

- `block_1.md` — теория вероятности и логика;
- `block_2.md` — решения задач на Python;
- `block_3.md` — решения задач на SQL;
- `block_4.md` — статистика и АБ-тесты;
- `block_5.md` — базовые задачи по ML.

Во всех блоках приведены краткие и понятные решения с пояснениями.  
В Python-блоке дополнительно приложен текст кода и дана оценка сложности алгоритмов.
